🛠️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/leetcode-java.git
   ```
2. Navigate into the folder:

   ```bash
   cd leetcode-java
   ```
3. Compile and run any solution:

   ```bash
   javac ProblemName.java
   java ProblemName
   ```
